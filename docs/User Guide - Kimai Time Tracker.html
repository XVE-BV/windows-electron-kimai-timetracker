Kimai Time Tracker - User Guide

Version: 1.0.0
Last Updated: January 2025

What is this app?

Kimai Time Tracker is a desktop application that helps you log your working hours to the Kimai time tracking system. It sits in your system tray and makes it easy to start and stop timers for different projects.

Installation

Step 1: Run the installer

Double-click on "Kimai Time Tracker-1.0.0 Setup.exe"

The app will install automatically - no administrator rights needed.

Step 2: Launch the app

After installation, the app starts automatically. You can also find it in your Start Menu under "Kimai Time Tracker".

Step 3: Find the tray icon

Look for the blue Kimai icon in your system tray (bottom-right corner of your screen, near the clock). Click it to open the app.

First-Time Setup

Configure your Kimai connection

  1. Click the tray icon to open the app
  2. Click "Settings" at the bottom
  3. Enter your Kimai details:
    • Kimai API URL: https://clockit.xve-web.eu/api
    • API Token: Your personal API token (get this from Kimai website)
  4. Click "Test Connection" to verify it works
  5. Click "Save Settings"
How to get your Kimai API Token:
Log in to Kimai web interface, go to your profile, and find the API section to generate a token.

Jira Setup (Optional)

If you want to log time to Jira tickets, you need to configure Jira in Settings.

Getting your Jira API Token

  1. Go to https://id.atlassian.com/manage-profile/security/api-tokens
  2. Log in with your Atlassian account
  3. Click "Create API token"
  4. Give it a name like "Kimai Time Tracker"
  5. Click "Create"
  6. Copy the token immediately (you won't see it again!)
Recommended: Use scoped API tokens

If your organization supports API tokens with scopes (OAuth 2.0), select only these scopes:
  • read:jira-work - to view and search issues
  • write:jira-work - to log time to issues

This limits what the app can do and is safer than full access tokens.
Security Warning:

Standard API tokens (from id.atlassian.com) have full access to your account - they cannot be scoped. Keep them safe:
  • Never share your API token
  • Don't send via email or chat
  • Revoke immediately if compromised
  • Use a separate token for each app

Configuring Jira in the App

  1. Open Settings in the app
  2. Scroll to the Jira section
  3. Enable Jira integration
  4. Enter your details:
    • Jira URL: https://your-company.atlassian.net (ask your admin)
    • Email: Your Atlassian account email
    • API Token: The token you just created
  5. Click "Test Jira" to verify
  6. Click "Save Settings"

Daily Usage

Starting a Timer

  1. Click the tray icon to open the app
  2. Select a Customer
  3. Select a Project
  4. Select an Activity
  5. (Optional) Add a description of what you're working on
  6. Click the green "Start Timer" button

Stopping a Timer

  1. Click the tray icon to open the app
  2. Click the red "Stop Timer" button
  3. Your time is automatically saved to Kimai

Using with Jira (Optional)

If Jira integration is enabled, you can link your time to a Jira ticket:

  1. Click "Link Jira ticket..."
  2. Select your ticket from the list
  3. Start your timer as normal
  4. When you stop, time is logged to Kimai
  5. If "Auto-log time to Jira" is enabled in Settings, time is also logged to Jira
Note: Auto-logging to Jira is disabled by default. Enable it in Settings under Jira Cloud if you want time automatically logged to Jira when stopping the timer.

Work Session Tracking

The app can remind you when you forget to track time:

ButtonWhat it does
StartBegin your work day - app will remind you if you're not tracking
PauseTaking a break - reminders are paused
StopEnd your work day - no more reminders

Adding Manual Time Entries

If you forgot to start the timer, you can add time manually:

  1. Click "Manual Entry" at the bottom of the app
  2. Select the date, start time, and end time
  3. Choose your project and activity
  4. Click "Save Entry"

Troubleshooting

ProblemSolution
App won't connect to KimaiCheck your API URL and token in Settings. Make sure you're connected to the internet.
Can't see my projectsYour Kimai account may not have access to those projects. Contact your administrator.
Timer won't startMake sure you've selected a Customer, Project, AND Activity.
Jira login failedCheck your Jira email and API token in Settings.
Important: Don't share your API tokens with anyone. They give full access to your Kimai account.
